# 开发规范

2022 年 10 月 30 日后，项目使用 TS + Vue3 + Vite 进行开发。

其中，Vue SFC 组件使用  `<script setup>` 写法。

```vue
<script setup lang="ts">
…
</script>
```

## 命名规则

- 组件名称 : 大驼峰
- 函数    : function
- 函数命名 : 大驼峰
- 私有变量 : 小驼峰
- 公有变量 : 小驼峰
- 常量    : 全大写，单词之间使用 `_` 分隔
- 模板    : 大驼峰

## 注释规则

- 尽可能详细
